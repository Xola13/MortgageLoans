 ReadMe file of the Mortgage loan calculator

* The program written on the source code is the mortgage calculator.

* It allows the user to input the amount of loan they took , the amount of interest rate they were charged and the number of years they were set to pay back the loan.

* The calculator will then output the amount a person should be paying monthly.

* It also give out the the total interest the borrower must pay over the period of the loan.

* The total amount a borrower should pay over the period of the loan.

* The program will also display out the Amortization Schedule :
 
 - Which will indicate the monthly payments 
 - Principal , interest & also a Remaining balance a borrower should pay.
